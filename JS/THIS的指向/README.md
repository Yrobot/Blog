## this的建立
一般函数在预编译过程中生成的AO上下文  
AO中除了函数内的一些属性和函数参数外，还有`arguments`和`this`(箭头函数没有)。  
所以函数（除了箭头函数）预编译过程会生成`this`隐式对象，并 __默认指向window。__
